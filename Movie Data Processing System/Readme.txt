Movie Data Processing System (C)

This C program reads semicolon-separated movie data from a file (`movie.txt`), stores it in dynamic arrays, and provides menu-driven querying and analysis.

Features
- Budget array (sorted by year and budget)
- Name array with genres and directors
- Dynamic memory allocation and management
- Menu-driven interface with 8 options:
  1. List Budget array
  2. List Name array
  3. List all genres
  4. List movies by year
  5. List movies by rating
  6. Show detailed info of a movie
  7. Show genre frequency
  8. Exit

Usage
1. Compile: `gcc MovieDataProcessingSystem.c -o program`
2. Run: `./program` (Linux/Mac) or `program.exe` (Windows)
3. Ensure `movie.txt` is in the same folder

Skills Demonstrated
C Programming, File Handling, Dynamic Memory, Structs, String Parsing, Data Structures, Menu-driven Interface
